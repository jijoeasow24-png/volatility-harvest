# VolatilityHarvest v2 — Enhanced Web App

## What's New in v2

### Monthly Action Plan (Dashboard hero)
A step-by-step checklist that tells you exactly what to do each month — with checkboxes to mark tasks done.

### Auto-Calculated Portfolio Value
No more manual entry. Portfolio value is automatically computed from your positions' live prices.

### Live Sparklines
30-day mini charts next to every position in Signals, Portfolio, and Markets tabs.

### Dark/Light Theme Toggle
Sun/Moon button in the header switches themes instantly.

### 52-Week High/Low
See where each stock sits in its 52-week range on the position detail view.

### Monthly Performance History
Visual bar chart tracking your portfolio value month over month.

### CSV Export
Download your full portfolio data as a CSV from the Portfolio tab.

### Signal Notifications
Banner alerts when stop-loss triggers fire on any position.

## Deploy / Update

Since your repo is already connected to Vercel, just replace the files in your GitHub repo:

1. Download and unzip this file
2. Go to your GitHub repo: github.com/jijoeasow24-png/volatility-harvest
3. Delete all existing files (or replace them)
4. Upload the new files from this zip
5. Vercel will auto-deploy within 60 seconds
